Crawler Finalizing Phases (PI Web API Indexed Search)

Finalizing

Finalizing is only performed for AF sources.

Phase 0

This phase indicates the crawler has not begun finalizing.

Phase 1

During this phase the crawler calculates the word counts within each search item.

Phase 2

During this phase the crawler calculates path fragments based on all the parent references of an element.

Phase 3

During this phase the crawler performs special handling of paths where elements have a direct reference from the root.

Phase 4

During this phase the crawler handles the multiple paths that each search item may have. The estimate is based on the current pass of the search items. At the beginning of each pass of the search items, the estimate is updated to the working total. The number of passes is unknown so the total is just an estimate.
Note: The number of operations needed to complete this phase grows at a very quick rate and each item takes longer to process the closer this phase is to completion.

Phase 5

This phase indicates the crawler has finished finalizing.
